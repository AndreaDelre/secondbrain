# Improvements - 2025-12-31

✦ Voici les 5 règles d'or issues de notre session, prêtes à être
  intégrées dans mon contexte (ou ton fichier de consignes) pour
  rendre nos futures interactions encore plus fluides :

  1. Règle de la "Shortlist" Sacrée (Today)
   * Consigne : Ne JAMAIS ajouter automatiquement de nouvelles
     tâches importées (ex: Google Tasks) ou détectées directement
     dans la section "Today".
   * Comportement attendu : Les nouvelles tâches vont dans leur
     section "Projet" respective ou dans une "Inbox". L'entrée
     dans "Today" se fait uniquement sur validation explicite de
     l'utilisateur.

  2. Pas de "Hallucination de Procédure" (Travaux/Technique)
   * Consigne : Sur des sujets d'expertise précise (Bricolage,
     Travaux), ne pas inventer de checklist "standard" ou
     "générique".
   * Comportement attendu : Créer la section vide et demander :
     "Quelles sont les étapes spécifiques que tu as prévues ?".
     Attendre la dictée de l'utilisateur avant d'écrire.

  3. Architecture "Dashboard" vs "Satellite"
   * Consigne : Garder le fichier _TODO.md lisible et synthétique.
   * Comportement attendu : Dès qu'un projet nécessite du détail
     (PRD, Liste de travaux pièce par pièce), créer un fichier
     dédié (ex: TRAVAUX.md, GEMINI_ASSISTANT.md) et ne laisser
     qu'un lien et la prochaine action dans le _TODO.md.

  4. Rituel de Clôture (Clean Archiving)
   * Consigne : Ne pas laisser s'accumuler les tâches finies dans
     le fichier principal.
   * Comportement attendu : Lors de la revue du soir, déplacer
     systématiquement les tâches terminées ("Done") vers un
     fichier journalier unique dans le dossier
     logbook/YYYYMMDD-logbook.md. Le _TODO.md doit toujours être
     prêt pour "Demain".

  5. Posture "Product Manager"
   * Consigne : Face à un projet flou, ne pas lister bêtement des
     tâches.
   * Comportement attendu : Basculer en mode "Interview" pour
     challenger la vision, les KPIs et les contraintes techniques
     (comme on l'a fait pour le PRD Gemini) avant de produire le
     moindre plan d'action.
